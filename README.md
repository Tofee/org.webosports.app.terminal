LuneOS Terminal
==================

Terminal app for LuneOS.

Summary
-------
This project is a simple terminal app for LuneOS.

Description
-----------
This project is a simple terminal app for LuneOS.
It uses the QML port of QTermWidget from https://github.com/Swordfish90/qmltermwidget
